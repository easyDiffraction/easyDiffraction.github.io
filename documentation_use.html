3.2. Data processing workflow

This section describes all the steps in the data processing with easyDiffraction, starting form the opening of input data files and ending with exporting of calculated parameters.

3.2.2. Load experimental data

Main window

The application's main window contains the following tabs:

  1. Plot View tab shows the experimental data plots.
  2. Table View tab contains a table with the extracted data.
  3. Text View tab displays the content of the data file.

An interaction with the plot is performed as follows:

  • Show coordinates: Hover mouse pointer.
  • Zoom in: Left mouse button.
  • Reset: Right mouse button.

Sidebar

The application's sidebar contains the following groups:

  • Data explorer: Shows experimental data file name and directory.
  • Instrument: Contains information about neutron scattering instrument.
  • Peak profile: Contains information about peak profile.
  • Misc: Contains miscellaneous information.

Supported file formats

A typical powder diffraction data file is shown below. #wavelength 0.84
#field 1.0
# ttheta      IntUP     sIntUP    IntDOWN   sIntDOWN
    4.00  465.80000  128.97000  301.88000  129.30000
    4.20  323.78000  118.22000  206.06000  120.00000
    4.40  307.14000  115.90000  230.47000  116.53000
    4.60  172.13000  115.84000  236.52000  113.74000
    ...

  • wavelength: Experimental neutron wavelength.
  • field: Applied magnetic field.
  • ttheta: Angular position of the Bragg peak.
  • IntUP: Intensity of scattered beam of neutrons polarized with spin "UP".
  • sIntUP: Standard deviation of IntUP.
  • IntDOWN: Intensity of scattered beam of neutrons polarized with spin "DOWN".
  • sIntDOWN: Standard deviation of IntDOWN.
