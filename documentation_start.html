2. Getting Started

This chapter is about getting started with easyDiffraction.

2.1. Installing easyDiffraction

easyDiffraction works across operating systems. You can download precompiled binaries of easyDiffraction Installer for macOS, Windows and Linux (soon) operating systems, or you can get the source code and build the program yourself.

Windows

  • Visit the easyDiffraction download page.
  • Choose Download for Windows.
  • Unzip the downloaded easyDiffractionInstaller zip file.
  • Double-click easyDiffractionInstaller.exe and follow the instructions given by the installer.

macOS

  • Visit the easyDiffraction download page.
  • Choose Download for macOS.
  • Unzip the downloaded easyDiffractionInstaller zip file.
  • Click easyDiffractionInstaller.app and follow the instructions given by the installer.

Linux

  • Visit the easyDiffraction download page.
  • Choose Download for Linux.
  • Unzip the downloaded easyDiffractionInstaller zip file.
  • Double-click easyDiffractionInstaller and follow the instructions given by the installer.

2.2. Launching easyDiffraction

After the program has been installed, you can run the graphical user interface version of easyDiffraction.

Windows

  • Double-click the easyDiffraction icon on the Desktop or
  • Select easyDiffraction from the Start Menu or
  • Navigate to the installation folder with Windows Explorer and double-click easyDiffraction.exe.

macOS

  • Double-click the easyDiffraction icon on the Launchpad or
  • Navigate to the installation folder with Finder and double-click easyDiffraction.app.

Linux

  • Double-click the easyDiffraction icon on the Desktop or
  • Navigate to the installation folder with file manager and double-click easyDiffraction.

2.3. Uninstalling easyDiffraction

You can uninstall easyDiffraction from macOS, Windows or Linux any time.

Windows

  • Open Windows Explorer and navigate to the folder containing installed easyDiffraction.
  • Double-click easyDiffractionUninstaller.exe and follow the instructions given by the uninstaller.

macOS

  • Open Finder and navigate to the folder containing installed easyDiffraction.
  • Click easyDiffractionUninstaller.app and follow the instructions given by the uninstaller.

Linux

  • Open file manager and navigate to the folder containing installed easyDiffraction.
  • Click easyDiffractionUninstaller and follow the instructions given by the uninstaller.
