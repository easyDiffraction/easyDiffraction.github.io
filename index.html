Making diffraction data analysis and
modelling easy

Currently available as a prototype: Version 0.3.2 (27-09-2019)

easyDiffraction is a scientific software for modelling and analysis of the neutron diffraction data, namely simulation of diffraction pattern based on structural models and refinement of their parameters. It is similar to other crystallographic programs such as FullProf, Jana, ShelX, GSAS, etc.

Cross-platform

easyDiffraction works across operating systems: Mac, Windows, or Linux (Ubuntu).

Easy to use

Intuitive tabbed interface helps to speed up the diffraction data modelling and analysis.
