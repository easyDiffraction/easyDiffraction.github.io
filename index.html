Making diffraction data analysis and
modelling easy

Currently available as a prototype: Version 0.3.5 (23 Oct 2019)

easyDiffraction is a scientific software for modelling and analysis of the neutron diffraction data, e.g., simulation of diffraction pattern based on structural model and refinement of its parameters. easyDiffraction is similar to crystallographic programs like FullProf, Jana, GSAS, ShelX, etc. However, unlike them, easyDiffraction is based on various external crystallographic libraries, such as CrysPy (more will be added soon) to cover different functionality within the single intuitive and user-friendly graphical interface. Everything is included

easyDiffraction is distributed as an all-in-one package that has no external dependencies and can be installed with only a few clicks.

Easy to use

Intuitive tabbed interface with build-in step-by-step user guides, and a clear workflow help to speed up the data modelling and analysis.
Learn more

Live update of calculations

You can modify any parameter manually or using slider, and the simulated curve is automatically recalculated in real time.
Learn more

Various techniques

We just started with classical 1D polarized neutron powder diffraction at constant wavelength. Unpolarized, single-crystal, 2D, Time-of-Flight, and so on, will be covered later.
Learn more

CIF-based project format

Human-readable data format CIF (Crystallographic Information File) is used for the project with the specs of International Union of Crystallography, where possible.
