1. Introduction

easyDiffraction is a scientific software for modelling and analysis of the neutron diffraction data.

1.1. What is easyDiffraction for?

easyDiffraction allows to simulate diffraction pattern based on structural models and refine their parameters. It is similar to other crystallographic programs such as FullProf, Jana, ShelX, GSAS, etc.

1.2. easyDiffraction features

  • Free and open-source
    • Anyone is freely licensed to use easyDiffraction and the source code is openly shared on GitHub.
  • Cross-platform
    • easyDiffraction works across operating systems. You can download precompiled binaries of easyDiffraction Installer for macOS, Windows and Linux operating systems, or you can get the source code and build the program yourself.
  • Multi-functional
    • easyDiffraction integrates existing crystallographic libraries, such as CrysPy (more will be added soon), to cover different functionality.
  • Everything is included
    • easyDiffraction is distributed as an all-in-one package that has no external dependencies and can be installed with only a few clicks.
  • Easy to use
    • Intuitive tabbed interface with build-in step-by-step user guides, and a clear workflow help to speed up the data modelling and analysis.
  • Live update of calculations
    • You can modify any parameter manually or using slider, and the simulated curve is automatically recalculated in real time.
  • Various techniques
    • We just started with classical 1D polarized neutron powder diffraction at constant wavelength. Unpolarized, single-crystal, 2D, Time-of-Flight, and so on, will be covered later.
  • CIF-based project format
    • Human-readable data format CIF (Crystallographic Information File) is used for the project according to the specs of International Union of Crystallography, where possible.
