1. Introduction

easyDiffraction is a scientific software for modelling and analysis of the neutron diffraction data.

1.1. What is easyDiffraction for?

easyDiffraction allows to simulate diffraction pattern based on structural models and refine their parameters. It is similar to other crystallographic programs such as FullProf, Jana, ShelX, GSAS, etc.

1.2. easyDiffraction features

  • Cross-platform
    • It works across operating systems. You can download precompiled binaries of easyDiffraction Installer for macOS, Windows and Linux operating systems, or you can get the source code and build the program yourself.
  • Easy to use
    • Intuitive tabbed interfaces of the program main window and sidebar help to speed up the diffraction data modelling and analysis.
